# Demo Video Recording Guide

Record a 3–6 minute screen capture showing the full TeamFlow flow.

## Suggested script

1. **Intro (15s)** – Show GitHub README briefly; open http://localhost:3000  
2. **Admin** – Login `admin@teamflow.local` / `Password123!`  
   - Dashboard stats  
   - Users page: create or change a role  
3. **Project Manager** – Logout; login `pm@teamflow.local`  
   - Create a project  
   - Add members  
   - Create tasks and assign them  
4. **Team Member** – Logout; login `alex@teamflow.local`  
   - Open assigned task  
   - Update status/progress  
   - Add a comment  
5. **Close** – Mention stack (Next.js, Express, MySQL) and that docs are in `/docs`

## Export

Upload to Drive/YouTube (unlisted) and put the link in your submission email / README.
